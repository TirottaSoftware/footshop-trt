# Footshop TRT
An E-commerce demo website for a sneakers store business.

## How to use 

Visit [TRT Footshop](http://footshop-trt.freecluster.eu/)
1. Browse sneakers at the men & women sections 
2. Or check out popular products at the home page
3. Check a particular product's information by clicking on its icon on the store page
3. Login or create an account 
4. Add/Remove products to your cart once you are logged in.

## Admin panel
Login as an admin user with the following credentials:
>username:  admin \
>password: admin

As an admin user, you can:

1. Retrieve general information about:
	* Total count of registered users
	* Total count of emails registered for the newsletter
	* Total count of products in the store
	* Total revenue
2. Manage products:
	* View total count of male & female products in the store
	* Add a product 
	* Remove a product
3. Manage users
	* Remove a user
	* Grant admin privileges to a user
	* View the user's total purchases and total products purchased
